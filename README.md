# 3D BRAIN MRI
Web application rendering human brain MRI in 3D.

Work in Progress - Code Refactoring and converting it to React

## Usage
In a terminal run:

```
git clone https://github.com/Tejaswini515/3D-Brain-MRI.git
cd 3D-BRAIN-MRI
npm install
node server.js
```
Then app renders in localhost:8080
